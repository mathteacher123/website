## MODIFIED Requirements

### Requirement: Test Home Page

The system SHALL provide a home page variant accessible at /homes/test that displays all sections from copy/test.md using appropriate components.

#### Scenario: Hero section displays

- **WHEN** user navigates to /homes/test
- **THEN** the hero section displays the headline "Direct Checkout. More Sales. Zero Friction."
- **AND** the subheadline "Skip the cart, redirect to custom URLs, and streamline your WooCommerce checkout flow in minutes."
- **AND** includes the call-to-action button "Boost Conversions Now"

#### Scenario: Problem section displays

- **WHEN** user navigates to /homes/test
- **THEN** the problem section displays "The Problem: The Conversion Killer" content
- **AND** describes the conversion issues with WooCommerce cart flow

#### Scenario: Solution section displays

- **WHEN** user navigates to /homes/test
- **THEN** the solution section displays "The Solution: Frictionless Checkout" content
- **AND** explains direct checkout and custom redirects benefits

#### Scenario: Use cases section displays

- **WHEN** user navigates to /homes/test
- **THEN** the use cases section displays "Key Use Cases" content
- **AND** lists direct checkout, affiliate products, terms & flows, and complex funnels

#### Scenario: Features section displays

- **WHEN** user navigates to /homes/test
- **THEN** the features section displays "Features List" content
- **AND** lists per-product settings, custom button text, hide messages, AJAX support, and global redirects

#### Scenario: Compatibility section displays

- **WHEN** user navigates to /homes/test
- **THEN** the compatibility section displays "Compatibility & Tech Specs" content
- **AND** mentions WooCommerce Blocks, Dokan, WordPress Multisite, and optimization

#### Scenario: Social proof section displays

- **WHEN** user navigates to /homes/test
- **THEN** the social proof section displays testimonials and trust indicators
- **AND** includes satisfaction guarantee

#### Scenario: Pricing section displays

- **WHEN** user navigates to /homes/test
- **THEN** the pricing section displays pricing table with Single Site, 5 Sites, and Unlimited options

#### Scenario: FAQ section displays

- **WHEN** user navigates to /homes/test
- **THEN** the FAQ section displays frequently asked questions
- **AND** answers about variable products, redirects, and theme compatibility

#### Scenario: Final CTA section displays

- **WHEN** user navigates to /homes/test
- **THEN** the final CTA section displays "Stop losing sales to abandoned carts" content
- **AND** includes "Boost Conversions Now" call-to-action
