# Change: Update Plugin Page Fonts and Icons

## Why

The /homes/plugin page currently uses Astrowind's default Inter font and Tabler icons, but to match the original design from code.html exactly, it needs to use the same Google Fonts (Spline Sans, Noto Sans) and Material Symbols icons.

## What Changes

- Update /homes/plugin page to use Google Fonts from code.html (Spline Sans for display, Noto Sans for body)
- Replace Tabler icons with Material Symbols Outlined icons throughout the page
- Ensure all font sizes, weights, and icon usage match the original HTML exactly

## Impact

- Affected specs: home-pages
- Affected code: src/pages/homes/plugin.astro and related components
